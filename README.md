# Project Backend 
# 2026 Peminjaman Ruangan - Backend

Sistem backend untuk mengelola data master ruangan dan transaksi peminjaman menggunakan ASP.NET Core.

## Teknologi Utama
* **Framework**: ASP.NET Core 8.0
* **Database**: SQL Server / PostgreSQL
* **ORM**: Entity Framework Core

## Cara Menjalankan
1.  Clone repositori ini.
2.  Lakukan `dotnet restore`.
3.  Update database: `dotnet ef database update`.
4.  Jalankan aplikasi: `dotnet run`.
